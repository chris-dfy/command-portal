# NEXUS Command

Standalone hosted command portal for the NEXUS Runtime API.

## Configurable identity

- Internal app ID: command-portal
- Temporary display name: NEXUS Command
- Temporary short name: NEXUS
- Optional parent brand: Nexicron

Visible branding is configuration, not an architectural constant. Runtime contracts, routes, data models, security policies, and API clients must remain independent of future product naming.
