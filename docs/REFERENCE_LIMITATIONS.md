This reference work follows the [NEXUS Platform Constitution](architecture/NEXUS_Platform_Constitution.md).

The expected `nexicron-demo-factory` GitHub repository was not accessible during Phase 5X-B. Therefore the curated Replit design reference could not be inspected, and this repository does not claim access to it.

The UI was derived from the two available local references:

- the generated `command-portal-reference` React/Vite client and its Phase 5X-A NEXUS Experience Gateway contract;
- the current `nexus-command` React/Vite desktop frontend, particularly its executive status rail, operational panels, verification center, specialist maturity distinctions, responsive rules, and accessibility preferences.

This limitation affects visual comparison only. It does not weaken the closed route allowlist, server-only credential boundary, truth posture, or mode behavior.
